# jocker

## test

make sure you have dependencies and truffle installed

### install truffle

```bash
yarn global add truffle
# or
npm install -g truffle
```

### install dependencies

```
yarn
# or
npm install
```


### run test

```bash
yarn test
# or
npm run test
```

### run pure test

run test ignore migrations scripts

```
yarn pure_test
# or
npm run pure_test
```
